# Telegram бот с котиками

Данный бот присылает случайные картинки котиков каждые 5 секунд.
Изображения берутся с <b>thecatapi</b>.

## Технологии
<ul>
    <li>JavaScript</li>
    <li>Node.js 22</li>
    <li>telegraf 4.16.3</li>
    <li>dotenv 17.2.4</li>
</ul>

## Использование
<ol>
    <li>Собрать проект командой <code>npm install</code></li>
    <li>Добавить токен бота и chat_id в .env файл</li>
    <li>Запустить проект командой <code>npm run start</code></li>
    <li>Написать боту и ожидать</li>
</ol>

## Примечание

На момент использования проекта принцип работы API мог измениться.
